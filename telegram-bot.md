# Telegram Bot Logic — Better Prompt

## Цель
Лаконично превращать пользовательские промпты в улучшенные версии в стиле сайта.

## Переменные окружения
- `TELEGRAM_BOT_TOKEN` — токен бота. Хранить только в переменных окружения (Railway / локальный `.env`). Не коммитить.

## Поведение
1. `/start`
   - Отправить короткое приветствие без эмодзи.
   - Отправить изображение бренда (можно reuse hero-арт сайта).
   - Кнопка: `Отправить промпт`.

2. Получение промпта
   - Принять любой текст.
   - Ответить сообщением-статусом: `обрабатываем.`
   - Обновлять статус каждые ~0.7 сек: `обрабатываем..` → `обрабатываем...` → `обрабатываем.`

3. Ответ
   - Отправить готовый промпт в моноширинном блоке.
   - Добавить кнопку `Скопировать` или подсказку: «Нажмите на текст, чтобы скопировать».

## Тон
Сдержанный, деловой, короткие фразы, без лишних знаков.

## Пример приветствия
Better Prompt — улучшение формулировок за минуты.
Отправьте ваш промпт, и я верну усиленную версию.
